# Release Notes

## v1.0.0

- Three fictional vendor exceptions
- Structured extraction and exception classification
- Six-stage visible orchestration
- Pass, review, and blocked policy outcomes
- Missing-information stop condition
- Internal exception approval or return
- Editable response draft and separate outbound approval
- Event and decision log
- Local JSON export
- Loading, validation, error, blocked, pending, returned, and approved states
- Responsive desktop and mobile layouts
- Logic, repository, local browser, and deployed browser validation

The demo is deterministic, fictional, local-first, and performs no external action.
